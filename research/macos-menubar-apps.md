# Beautiful macOS Menu Bar Apps (Open Source)

Research on the most beautiful and well-designed macOS menu bar apps, with focus on open-source options.

---

## Menu Bar Managers (Hide/Organize Icons)

### Ice (Most Popular)
The most ambitious free Bartender alternative. Beautiful visual customization including:
- Solid and gradient tinting
- Shadow and border effects
- Custom shapes (rounded/split)
- Drag-and-drop arrangement
- Works great with notched MacBooks

**GitHub**: https://github.com/jordanbaird/Ice
**License**: GPL-3.0
**Requirements**: macOS 14+

### Hidden Bar
Ultra-lightweight, minimal design. Drag icons behind a divider to hide them. Simple and elegant.
- Available on Mac App Store
- MIT License

### Dozer
Minimalist approach - does one thing perfectly. Uses <50MB RAM and <0.5% CPU.

## System Monitors

### Stats
Free, native system monitor showing CPU, GPU, memory, disk, network, battery in real-time. Modular design lets you enable only what you need.

**GitHub**: https://github.com/exelban/stats
**Website**: https://mac-stats.com/

### iGlance
Highly customizable system monitor. Each component can be styled individually.

**GitHub**: https://github.com/iglance/iGlance

## Productivity Apps

### Itsycal
Tiny, beautifully designed calendar. Clean interface, integrates with Apple Calendar, highly customizable. Users describe it as "one of the handiest, most trouble-free utilities ever."

**GitHub**: https://github.com/sfsam/Itsycal
**Website**: https://www.mowglii.com/itsycal/
**License**: MIT

### Maccy
Native, minimalist clipboard manager. "Looks and feels exactly as you'd expect" - pure macOS aesthetic. Blazingly fast search.

**GitHub**: https://github.com/p0deje/Maccy
**Website**: https://maccy.app/
**License**: MIT

### Reminders MenuBar
SwiftUI-based reminders app with animations that "match macOS perfectly." Natural language parsing, instant iCloud sync, ~25MB memory usage.

**GitHub**: https://github.com/DamascenoRafael/reminders-menubar
**License**: GPL-3.0

**GitHub**: https://github.com/swiftbar/SwiftBar

## Key Design Patterns Observed

1. **Minimal footprint** - Best apps use <50MB RAM
2. **Native macOS design** - SwiftUI/AppKit for authentic look
3. **SF Symbols** - Using Apple's icon library for consistency
4. **Single purpose** - Do one thing well
5. **Customizable appearance** - Tinting, shapes, spacing options
6. **Keyboard shortcuts** - Quick access without mouse
